# Gemini AI

Gemini AI is an advanced web application designed to provide accurate and concise answers to any questions you have. Built with a focus on user experience and functionality, Gemini AI offers a range of features to enhance user interaction.

## Features

- **Smart Q&A System**: Get precise and relevant answers quickly.
- **Custom Face Logo**: Personalized with a custom face logo for unique branding.
- **Settings Feature**: Customize your experience with various settings options.
- **Recent Tabs**: Easily access your recent interactions and activities.
- **Help & Activity Tabs**: Navigate assistance resources and track your activity history effortlessly.

- ![Gemini Ai 1](https://github.com/vishnuemail96/Gemini-Ai/assets/95302271/18a7db6b-a00d-4358-991e-06eb9fdc82d1)


## Technologies Used

- **React**: For building the dynamic and responsive user interface.
- **HTML & CSS**: To structure and style the web application for a seamless user experience.
- **JavaScript**: For adding interactivity and enhancing functionality.
- **Vite**: To streamline the development process with faster builds and hot module replacement.
- **Node Modules**: For managing dependencies and enhancing functionality.

## Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/your-username/gemini-ai.git
   ```

2. **Install NPM packages**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```

## Usage

Once the development server is running, you can access Gemini AI in your web browser. The application allows you to:

- Ask questions and receive precise answers.
- Customize the interface using the settings feature.
- View recent interactions and activities.
- Access help resources and track activity history.
